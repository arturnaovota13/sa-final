Resistores e Código de Cores

Os resistores limitam a quantidade de corrente que flui através de um circuito. Cada resistor possui um valor de resistência medido em ohms (Ω), identificado pelo código de cores.

Código de Cores: Cada faixa de cor no resistor representa um número. Um resistor com faixas marron, preto e vermelho indica um valor de 1kΩ.

Diodo LED e Display de Sete Segmentos

Diodo LED: LEDs emitem luz quando a corrente flui através deles. São usados para indicar status em circuitos.

Display de Sete Segmentos: Compostos por sete LEDs organizados para exibir números e algumas letras.

O Tinkercad é uma ferramenta de simulação que permite projetar circuitos e programar Arduino virtualmente.

Exemplo do Projeto: Esse circuito faz simulação de um potênciometro que quando passa de 859 o led Vermelho acende.

Placa Arduino

O Arduino é uma plataforma de prototipagem baseada em hardware e software livre, permitindo criar uma variedade de projetos de automação e controle.

Programa na IDE Arduino

A IDE Arduino permite escrever, compilar e carregar código para a placa. Exemplo de código:

// C++ code
//
int sensor = 0;

void setup() {
pinMode(A0, INPUT);
pinMode(8, OUTPUT);
}

void loop() {
sensor = analogRead(A0); // Potênciometro.
if (sensor >= 859) {
digitalWrite(8, HIGH); // LED Liga.
} else {
digitalWrite(8, LOW); // LED Desliga.
}
delay(10); // Delay após execução do Código.
}

Entrada Analógica no Arduino

Entradas analógicas permitem ler valores variáveis, como a tensão de um sensor. A leitura é feita usando a função analogRead() que retorna valores entre 0 e 1023.

Entrada Digital no Arduino

Entradas digitais detectam se um sinal está em nível alto (HIGH) ou baixo (LOW), ideal para botões e chaves.

Saída Analógica no Arduino (PWM)

As saídas PWM permitem controlar intensidade de LEDs ou velocidade de motores usando analogWrite(), variando de 0 (0%) a 255 (100%).

Saída Digital no Arduino

Saídas digitais controlam o estado (ligado/desligado) de componentes. Um LED pode ser controlado com digitalWrite(pin, HIGH) e digitalWrite(pin, LOW).



Multímetro - O que é e para que Serve

O multímetro é um instrumento que mede tensão, corrente e resistência. Essencial para diagnósticos e testes de componentes em circuitos.

Unidades de Medida no Multímetro

  • Voltagem (V): Medida da diferença de potencial entre dois pontos.
  • Corrente (A): Quantidade de carga elétrica passando por um ponto em um segundo.
  • Resistência (Ω): Oposição ao fluxo de corrente.
